HTML Structure

This theme is responsive layout. All the content is wrape up with the page-wrapper class. The basic structure contains :

<header></header> - Logo and navigation

<footer></footer> - Copyright info

All other contents separate by it's own section with HTML section element or div.

<section></section> or <div></div> - All other contents

SASS Files and Structure

All SASS file are located in the "sass" folder of the template. For changing the theme style you can change the sass files as you want. If you want to change the entire theme color you can change the $theme-primary-color, $theme-secondary-color variable value which is located in the helper folder _variables.scss file

Javascripts

All JavaScript / jQuery plugins are located in the "js" folder of the template. You do not need to edit these file. script.js file is the main javascript file for the theme behavior located in the same folder. You can edit it if you want.

RSVP

For setting the contact mail open the item folder, find mail.php file and open it. Then replace the $to vaiable value with your email address. All set.
